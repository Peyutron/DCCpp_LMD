# DCCpp-LMD

DCCpp LMD Librería DCC++ para Arduino
Diseñada para placa Arduino MEGA con pantalla Oled y multiples puertos seriales:

<strong>
*- Pantalla Oled con librería U8G2lib

*- Serial → PC

*- Serial1 → Wifi

*- Serial2 → Bluetooth

*- Serial3 → Serial Auxiliar

*- Sonido en pin 6
</strong>
