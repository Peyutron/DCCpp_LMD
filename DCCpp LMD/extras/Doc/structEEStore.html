Detailed Description

This structure/class stores accessories data in EEPROM.

The data is saved from byte 0 of EEPROM. The content is :

0--------------------------------------+----------+---------+---------+
| EEStoreData(see EEstoreData content) | Turnouts | Sensors | Outputs |
+--------------------------------------+----------+---------+---------+
